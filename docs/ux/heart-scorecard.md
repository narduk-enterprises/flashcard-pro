# HEART Scorecard (Header + Primary Navigation)

| Dimension | Goal | Signal | Metric |
|---|---|---|---|
| Happiness | Reduce friction in header interactions | Fewer repeated clicks/confusion events | Lower repeated logout/menu toggle attempts per session |
| Engagement | Encourage movement to high-value routes | More discover/study transitions from header | CTR on header links |
| Adoption | Help new users understand top-level nav quickly | First-session route transitions happen earlier | Time-to-first-nav-click |
| Retention | Keep return users in flow with reliable controls | Stable use of same nav controls across visits | Returning session nav usage rate |
| Task Success | Ensure users can navigate/logout without ambiguity | Successful route changes and logout completion | Navigation success rate + logout success rate |
